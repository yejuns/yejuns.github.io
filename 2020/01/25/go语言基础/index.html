go语言基础

2020-01-25 go语言 评论

常用命令

go run test.go 运行go文件

go build test.go 编译相关go文件（编译成功后，会生成一个2进制文件）

应用程序的入口

  1. 必须是main包，package main
  2. 必须是main方法， function main
  3. 文件名不一定是 main.go

go语言的类型

  1. 常用的类型bool、int、float、byte等，不一样的是，还存在 complex 复数类型
  2. go语言不允许隐式类型转换
  3. go语言存在指针，但是不支持指针运算
  4. String是数值类型，初始化时是 0 ，而不是 null go语言的arr、slice、map都收到 python的影响，所以有些地方比较类似

    数组

Go 语言中的数组是一种 值类型，当把一个数组赋值给另一个时，需要再做一次数组内存的拷贝操作，比如

这样两个数组就有了不同的值，在赋值后修改 arr2 不会对 arr1 生效。

所以在函数中数组作为参数传入时，如 func1(arr2)，会产生一次数组拷贝，func1 方法不会修改原始的数组 arr2。

如果你想修改原数组，那么 arr2 必须通过&操作符以引用方式传过来，例如 func1(&arr2）。

由于切片比数组要好用，实际项目中我们常使用的是切片

切片

1.切片（slice）是对数组一个连续片段的引用（该数组我们称之为相关数组，通常是匿名的），所以切片是一个引用类型（因此更类似于 C/C++ 中的数组类型）
2.切片的长度可以在运行时修改，最小为 0 最大为相关数组的长度：切片是一个 长度可变的数组。

函数

Go 里面有三种类型的函数

  1. 普通的带有名字的函数
  2. 匿名函数或者lambda函数
  3. 方法

除了main()、init()函数外，其它所有类型的函数都可以有参数与返回值，而且返回值可以是多个值

函数也可以以申明的方式被使用，作为一个函数类型, 比如：type myFunc func(int, int) (int,int)

go的特殊语法

go语言交换两个变量

a,b = b,a

相比较其他语言需要借助中间变量，go语言的语法更加简单

测试多返回值函数的错误

Go 语言的函数经常使用两个返回值来表示执行是否成功：返回某个值以及 true 表示成功；返回零值（或 nil）和 false 表示失败（第 4.4 节）。
当不使用 true 或 false 的时候，也可以使用一个 error 类型的变量来代替作为第二个返回值：成功执行的话，error 的值为 nil，否则就会包含相应的错误信息

空白符 _

空白符用来匹配一些不需要的值，然后丢弃掉。

延迟函数 defer

defer 允许我们推迟到函数返回之前（或任意位置执行 return 语句之后）一刻才执行某个语句或函数，类似其他语言中的 finally

为什么要在返回之后才执行这些语句？因为 return 语句同样可以包含一些操作，而不是单纯地返回某个值，比如关闭数据库连接，关闭一些锁。
